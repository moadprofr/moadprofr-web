# 🚀 Moadprofr Web Platform

Plateforme web officielle du projet **Moadprofr**.

---

## 📌 Description

Moadprofr est une plateforme web moderne qui permet aux utilisateurs de :

* 🔐 Créer un compte et se connecter
* 👤 Gérer leur profil
* 💎 Accéder à du contenu Premium sécurisé
* ⚡ Profiter d'une expérience fluide en temps réel

---

## 🛠️ Technologies utilisées

* 🌐 Frontend : Flutter Web / HTML / CSS / JavaScript
* ⚡ Backend : Supabase
* 🗄️ Base de données : PostgreSQL
* 🔒 Authentification : Supabase Auth
* 🚀 Hébergement : Netlify / Vercel

---

## ✨ Fonctionnalités

* ✅ Authentification (Email / Google)
* ✅ Système Premium (activation via base de données)
* ✅ Mise à jour en temps réel (Realtime)
* ✅ Protection des données avec RLS
* ✅ Upload de fichiers sécurisé
* ✅ Gestion des utilisateurs
* ✅ Dashboard Admin

---

## 🔐 Sécurité

* Row Level Security (RLS)
* Accès limité aux données utilisateur
* Vérification côté serveur
* Protection des contenus Premium

---

## 📦 Installation

```bash
git clone https://github.com/your-username/moadprofr-web.git
cd moadprofr-web
```

---

## ⚙️ Configuration

Créer un fichier `.env` :

```env
SUPABASE_URL=your_url
SUPABASE_ANON_KEY=your_key
```
